Mac 安装软件记录

发表于2025-03-29|更新于2025-04-07|Mac
|总字数:1.6k|阅读时长:5分钟|浏览量:

这里整理了一些在 macOS 上实用的工具，涵盖开发、生产力、系统管理等多个方面，并附上简要介绍和下载链接。

  • 剪贴板工具

    快捷键：Ctrl + Option + Command + C

    剪贴板管理工具可以帮助你记录并快速访问历史复制的内容，提高工作效率。

    • Maccy
      轻量级的 macOS 剪贴板管理工具，支持快捷键调用，搜索复制历史，并且免费开源。
    • EcoPaste
      另一个 macOS 剪贴板工具，专注于隐私保护，支持剪贴板自动清理。

    截图软件

    快捷键：Ctrl + Command + A

    • Snipaste
      强大的截图工具，支持贴图功能，可以把截图直接固定在屏幕上，方便参考。

    翻译工具

    快捷键：Ctrl + 2

    • 欧路词典
      macOS 端强大的词典工具，支持多种翻译引擎，支持划词翻译和生词本管理。

    音频处理

    • Audacity
      免费开源的音频编辑软件，支持多轨录音、剪辑、降噪等功能。

    抓包工具

    抓包工具可以用于网络请求分析、调试 API，适合开发者和安全研究人员。

    • Charles
      专业的 HTTP 代理工具，支持网络请求拦截、修改、重放等功能。
    • Proxyman
      现代化的 macOS 抓包工具，支持 HTTPS 解密，界面友好，适合 iOS/Android 开发者。

    三维模型

    • Blender
      强大的开源 3D 建模和动画制作工具，支持建模、渲染、动画、物理模拟等功能。

    AI 相关

    人工智能助手可以用于代码编写、文档撰写、调试等多种任务。

    • ChatGPT
      OpenAI 旗下的 AI 助手，适用于对话、生成代码、写作等任务。
    • Claude
      Anthropic 开发的 AI 助手，专注于安全性和长文本处理。
    • Gemini
      Google 开发的 AI 模型，适用于搜索增强、代码生成等。
    • Grok
      由 X（前 Twitter）开发的 AI 助手，整合社交媒体信息。
    • Poe
      由 Quora 推出的 AI 平台，集成多个大语言模型（GPT、Claude 等）。

    科学上网工具

    • ClashX Pro
      适用于 macOS 的代理工具，支持多种协议，适合科学上网和流量管理。
    • Shadowrocket
      iOS 平台的代理工具，支持多种协议，需付费购买。

    视频流分析工具

    用于分析视频编码、比特率、帧率等信息。

    • CodecVisa
      专业的视频流解析工具，支持 H.264、H.265 等格式。
    • StreamAnalyzer
      高级视频流分析工具，支持多种编解码格式。
    • StreamEye
      视频流分析软件，可用于调试视频质量问题。
    • YUView
      开源的视频分析工具，支持 YUV 视频查看与分析。
    • YUVToolkit
      轻量级的 YUV 视频查看工具。

    数据库管理

    • DB Browser for SQLite
      适用于 SQLite 数据库的可视化管理工具。
    • DBeaver
      通用数据库管理工具，支持 MySQL、PostgreSQL、MongoDB 等。

    Git 工具

    • Fork
      现代化的 Git 客户端，支持 GitHub、GitLab 等平台。
    • Sourcetree
      Atlassian 出品的 Git 可视化管理工具。

    浏览器

    • Google Chrome
      兼容性最强的浏览器，支持大量开发者扩展。

    二进制工具

    • Hex Fiend
      macOS 上强大的十六进制编辑器，适合分析二进制文件。
    • ImHex
      现代化的十六进制编辑器，支持高级模式和数据分析。

    视频播放器

    • IINA
      macOS 上最佳的开源视频播放器，基于 mpv，支持多种格式。

    终端工具

    • iTerm2
      macOS 上最强大的终端工具，支持分屏、标签页等高级功能。

      • oh-my-zsh（推荐插件）

    JSON Model 生成

    • JsonToModel
      macOS 上的 JSON 解析和 Model 生成工具。

    解压缩工具

    • Keka
      macOS 上强大的解压缩工具，支持 7z、RAR、ZIP 等格式。

    风扇控制

    • Macs Fan Control
      监控和调整 Mac 风扇转速，降低温度和噪音。

    iOS 崩溃符号化

    • MacSymbolicator
      iOS 开发者用于符号化崩溃日志的工具。

    远程工具

    • Microsoft Remote Desktop
      远程访问 Windows 设备的官方工具。
    • VNC Viewer
      远程桌面访问工具，支持 VNC 协议。

    RSS 阅读工具

    • Reader
      macOS 端强大的 RSS 订阅阅读器。
    • Folo
      现代化的 RSS 阅读工具，支持 iCloud 同步。

    文本处理

    • Notion
      笔记和知识管理工具，支持多人协作。
    • Typora
      最简洁的 Markdown 编辑器。
    • WPS Office
      兼容 Microsoft Office 的办公软件。

    电源管理

    • AIDente
      MacBook 充电管理工具，可以限制最大充电电量，延长电池寿命。

    下载工具

    • Downie
      强大的视频下载工具，支持 YouTube、Bilibili 等主流网站的视频下载。

    VPN 客户端

    • OpenVPN Connect
      OpenVPN 官方客户端，支持自建 VPN 连接。

    Android 设备控制

    • QtScrcpy
      基于 Scrcpy 的 Android 设备投屏与控制工具，支持无线连接和物理按键映射。

开发工具

  • Visual Studio Code
    微软出品的轻量级代码编辑器，支持丰富的扩展插件，适合多种开发环境。
  • Xcode
    Apple 官方的 iOS/macOS 开发工具，内置 Swift 和 SwiftUI 支持。

网盘工具

  • 百度网盘
    国内主流的云存储服务，支持大文件存储与分享。

命令行工具

  • trash
    命令行版 rm，可将文件移至回收站而不是直接删除。

  • clang-format
    代码格式化工具，支持 C++、Objective-C、Swift 等多种语言。

  • ffmpeg
    强大的音视频处理工具，可用于格式转换、压缩、剪辑等。

  • node.js
    JavaScript 运行时，适用于前端开发、服务器端开发。

    • Hexo：基于 Node.js 的静态博客框架，可快速生成个人博客。

  • ninja
    轻量级、高速的构建系统，适用于 C++ 等编译项目。

  • rbenv
    Ruby 版本管理工具，方便切换不同版本的 Ruby 运行环境。

    • Ruby：适用于 Ruby 开发，尤其是 Rails 项目。

  • Python
    强大的通用编程语言，适用于数据分析、自动化、AI 开发等。

  • Spatial

    是一款 macOS 命令行工具，主要用于解析和处理 Apple 的 .spatial 文件格式。.spatial 文件通常用于 Apple 设备的 3D 数据存储，例如 Apple Vision Pro 的空间视频或其他 3D 相关内容。
